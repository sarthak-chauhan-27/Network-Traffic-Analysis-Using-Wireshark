# Network-Traffic-Analysis-Using-Wireshark
Wireshark-based network traffic analysis lab investigating DNS, DHCP, TCP, HTTP/HTTPS, ICMP, and network troubleshooting techniques.

Wireshark Network Traffic Analysis Lab - 

Overview - 

This project demonstrates hands-on network traffic analysis using Wireshark in a Windows-based home lab environment. The objective was to capture, inspect, and analyze network traffic to gain a deeper understanding of network protocols, troubleshooting methodologies, and security monitoring techniques.

The project focuses on analyzing common network protocols and understanding how devices communicate across a network. Packet captures were examined to identify connection establishment processes, DNS resolution, DHCP address assignment, web traffic behavior, and network troubleshooting scenarios.

Objectives - 

Capture live network traffic using Wireshark
Analyze common network protocols
Understand packet structures and communication flows
Perform network troubleshooting activities
Develop packet analysis skills used in IT Support and SOC environments
Document findings and observations
